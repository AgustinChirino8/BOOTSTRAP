DEFINICION:

JavaScript es un lenguaje de secuencias de comandos que te permite crear contenido de actualización dinámica, controlar multimedia, animar imágenes y prácticamente todo lo demás. (Está bien, no todo, pero es sorprendente lo que puedes lograr con unas pocas líneas de código JavaScript).

Tipos de datos en JavaScript

Una de las características de JavaScript que también deberías conocer son los diferentes tipos de datos existen y que mencionaremos a continuación.

  • String: se utiliza para trabajar con cadenas de texto.
  • Boolean: se refieren a las entidades lógicas que pueden tener dos valores (verdadero y falso).
  • Number: se admiten solo datos numéricos.
  • Null: es cuando el valor asignado tiene valor nulo.
  • Symbol: son los datos que tienen valores únicos e inalterables.
  • Object: es un valor al que se puede acceder mediante un identificador.
